# Nextcent - Professional Landing Page

This is a modern landing page project designed for agencies and community management systems. The site features a clean and professional interface using modern layout practices.

##  Technologies Used
* **HTML5:** Semantic structure (header, main, section, article).

* **CSS3:** Custom styles and responsive design.

* **Google Fonts:** "Inter" font for modern typography.

* **SVG & PNG Assets:** Use of icons and vector graphics.


##  Project Structure
The project consists of the following key sections:

1. **Header & Navbar:** Navigation menu

2. **Hero Section:** Introduction with a call to action for registration.

3. **Clients Section:** Logo gallery of Fortune 500+ clients.
4. **Community Management:** Information cards about organizations, associations, and groups.

5. **About Section:** Information about the company's history (Pixelgrade).

6. **Stats Section:** Visual counter of community members and achievements.

7. **Footer Design Section:** Details about website design and management.
